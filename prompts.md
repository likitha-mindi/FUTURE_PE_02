# PROMPTS USED – UGC AD GENERATION

## Prompt 1 – Instagram UGC Ad Script
Act as a UGC content creator making a short Instagram reel.
Create a 30-second ad script for an online coding course named "CodeSprint Academy".

Target audience:
- College students
- Beginners with no coding background
- Career switchers

Structure:
Hook (first 3 seconds)
Problem
Solution
Social proof
Strong CTA

Tone:
Relatable, natural, slightly emotional, motivating.

## Prompt 2 – Multiple Hooks
Generate 10 scroll-stopping hooks for an online coding course.

The hooks must:
- Be short (1 sentence)
- Create curiosity
- Address career pain points
- Feel like spoken language
  
## Prompt 3 – YouTube Shorts Version
Create a YouTube Shorts UGC-style ad script for CodeSprint Academy.
Keep it energetic and motivational.
Focus on:
- High salary jobs
- Learning from scratch

End with a strong enrollment CTA.
