# FUTURE_PE_02  
## AI Content Marketing using UGC Ads  

### Internship Track:
Prompt Engineering – Future Interns  

## Project Overview  
This project demonstrates structured prompt engineering to generate high-converting UGC-style advertisement scripts for an online coding course.

The objective is to create:
- Scroll-stopping hooks
- UGC-style ad scripts
- Platform-specific ad variations
- Strong call-to-action messaging

All prompts, outputs, and reasoning are documented in this repository.

## Product Chosen  
**Product Name:** CodeSprint Academy  
**Type:** Online Coding Course  
**Target Audience:** College students, beginners, career switchers  
**Tone:** Relatable, motivational, authentic  

## Repository Structure  
- prompts.md → Structured prompts used  
- outputs.md → AI-generated ad scripts  
- explanation.md → Prompt strategy and logic  

## Learning Outcome  
- UGC ad structuring  
- Conversion-focused messaging  
- AI-driven content marketing workflows
